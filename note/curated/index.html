奇幻

魂環 / スピリットサークル

細膩，溫和，充滿宿命感又帶著一縷鄉愁味道的輪迴系作品。在猴區被稱為炎拳唯一指定解藥。
水上悟志老師講故事的功力了得，對情緒起伏的把握也妙到毫巔，總能從時間長河中的細枝末節處挖掘出真情實感，編織成環，然後再將之匯聚到一點集中爆發出來。
個人覺得優秀的作品就理應是這個樣子的。既有宏大背景所帶來的戲劇張力，又有心思細膩所帶來的情感共振，悲傷流淚後也會轉眼間破涕而笑。

鋼之鍊金術師 / 鋼の錬金術師

故事工整節奏緊湊，十本左右的內容幾乎全是主線完全沒有注水部分。應當被列入少年漫畫敘事教科書（沒有這種東西
除了穩紮穩打環環相扣的敘事外，本作最令人讚歎的是將設定中的鍊金術原理（循環、流動、一即是全全即是一）完完全全貫徹進了角色信條、情節發展、故事理念等整部作品的方方面面，就連完滿的收尾都像極了煉成陣基礎的圓。
不過論情感渲染以及歷史厚重感，個人認為還是FA動畫版更勝一籌。

運動

舞冰的祈願 / メダリスト

雙主角花樣滑冰作品。絕對的畫力和分镜能力能給人帶來無與倫比的沉浸感和震撼，台詞也是濃縮的雞湯震得心砰砰迴響。
甚至還讓筆者這種花滑知識為零的徹徹底底門外漢特地跑去百科上背了一大堆術語。不過作品內的講解已經足夠用心了即使完全不了解花樣滑冰也可以放心去看。
請大家在眼睛遭到ENGI的動畫改編污染前先來看看原作固定一下第一印象！

Webtoon

劍尊歸來 / 화산귀환

近幾年大量發生的重生流武俠Webtoon條漫，華山派高手死掉轉生到100年後的故事。但這篇是Webtoon武俠分類第一，搞不好也是Naver歷年所有作品中綜合實力最高的一作。
大工作室出品，文戲武戲以及顏藝水準都幾乎無可挑剔。有著恰到好處的光影與行雲流水般的分鏡，嚴肅與喜劇之間的平衡也把握得很出色。
有別於市面上其他個人主義龍傲天爽漫，本作主角開局就很強卻要肩負起重振門派的重任。因為個人實力已經沒有可提升的空間了於是作者另闢蹊徑，著墨於天才的侷限性以及門派意志延續的必要性，將重點聚焦於「華山派」這個整體而非主角一人。隨著晚輩們實力逐漸提升，主角也經歷了心境上的成長。
是少數能讓筆者體會到真正的武俠精神的韓國Webtoon漫。

狂魔重生記 / 광마회귀

還是重生流武俠作品，有著狂魔稱號的高手轉生的故事。和上面那部不同，這部比較吃電波，對上電波了就會覺得非常有意思，對不上會覺得主角和劇情莫名其妙。
故事有趣就在於，身為狂魔的主角真的有夠狂。他不僅行事張狂也毫無邏輯可言，只要想到什麼就會立刻付諸實行。對於身邊一眾常識人跟班來講，光是要試著跟上主角的腳步就已經費盡全力，更遑論追上他的思緒。
作品主要通過主角反覆無常的個性製造各種吐嘈與笑料來推進；看狂魔各種開掛發神經，然後配角吐槽，娛樂度極高！非常優秀的廁所讀物！

BL

死去的摯友向我表白了 / 死んだ親友に告白された話

去世很久的親友化身幽靈向主角告白的BL作品。身為死亡題材卻沒有想象中那麼沉重，非常平靜溫馨又充滿生活氣息，但回過頭來重看又有種淡淡的哀傷。
親近之人的離開不是一場暴雨，而是漫長的潮濕。在某個平凡的日子裡，當與那個人相關的記憶無意中被觸動時，那些共同度過的、看似微不足道但又難以忘懷的細節，就會像巨石一樣砸向心防。
兩個人未盡的話都用「打遊戲」來取代。既是彼此之間的聯結，也是難以宣之於口的心情。
